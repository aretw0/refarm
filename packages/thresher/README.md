# @refarm.dev/thresher

Thresher is Refarm's data extraction and harvesting engine. It is the sibling to **Sower**, together forming the complete ETL cycle.

## Role

Thresher "harvests" data from the Sovereign Graph and exports it into external formats (JSON-LD, SQLite, CSV).

See [packages/sower/ROADMAP.md](../sower/ROADMAP.md) for the shared ETL roadmap.
