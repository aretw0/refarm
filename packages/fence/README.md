# @refarm.dev/fence

Fence provides security and sandboxing utilities for the Refarm ecosystem. It ensures that plugins run in isolated environments and can only access the capabilities they are explicitly granted.

## Features

- **Sandboxing Contracts**: Defining the boundary between the host and WASM plugins.
- **Capability Isolation**: Enforcing least-privilege access to system resources.
- **Runtime Protection**: (Planned) Real-time monitoring of plugin behavior.

See [ROADMAP.md](./ROADMAP.md) for the path to formal sandboxing architectures.
