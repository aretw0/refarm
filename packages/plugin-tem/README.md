# @refarm.dev/plugin-tem

The Plugin-TEM integrates the Tolman-Eichenbaum Machine (TEM) reasoning engine into the Refarm ecosystem, enabling autonomous navigation and organization of the Sovereign Graph.

## Role

TEM provides "Structural Intelligence" — it learns the relational topology of the user's data and can proactively suggest connections or detect novelty without requiring central training.

See [ROADMAP.md](./ROADMAP.md) for the path to kernel-level intelligence.
