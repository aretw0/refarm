# Refarm Antenna - TypeScript Template

This is a template for building an Antenna (HTTP Gateway) plugin for Refarm using TypeScript and `jco`.

## Structure

- `src/index.ts`: The main Antenna processor.
- `package.json`: Build commands (`npm run build`).

## Building

1. Run `npm install`
2. Run `npm run build` to generate the WASM component.
