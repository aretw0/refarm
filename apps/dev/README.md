# @refarm.dev/studio (apps/dev)

Refarm Studio (also known as `apps/dev` or `refarm.dev`) is the primary management interface and integrated development environment (IDE) for the Refarm ecosystem.

## Features

- **Sovereign Graph Visualization**: Explore nodes and relationships in real-time.
- **Plugin Management**: Inspect, configure, and install plugins via the graph.
- **Dev Tools**: Real-time telemetry, logs, and metrics from the microkernel.

See [ROADMAP.md](./ROADMAP.md) for the strategic vision.
